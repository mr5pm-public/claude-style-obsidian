# 클로드 스타일 옵시디언 · Claude Style for Obsidian

> 매일 들여다보는 메모장을, **클로드(Claude) 그 느낌** 그대로.
> by 오후다섯씨 (MR_5PM) · **평생 무료**

명조/고딕체 본문 · 넓은 자간·줄간격 · 가운데 정렬 · 테두리 없는 깔끔한 표 · 다크 테마 · 클로드 주황(`#C96442`) 강조색.

AI가 만들어내는 마크다운(`.md`) 파일을, 옵시디언에서 **클로드 화면처럼 고급스럽게** 보고 쓰기 위한 CSS 스니펫입니다.

---

## 📥 30초 설치

1. 옵시디언 → **설정(Settings)**
2. **외형(Appearance)** 탭
3. 맨 아래 **CSS 스니펫**에서 `[폴더 열기]` 클릭
4. 열린 폴더에 이 저장소의 [`claude-style.css`](./claude-style.css)를 그대로 넣기
   - (오른쪽 파일 → `Raw` → 저장, 또는 저장소 전체 `Code → Download ZIP`)
5. 옆의 `[새로고침]` 한 번
6. 목록에 뜨면 **토글 ON** — 끝!

> **폰트**: 본문은 부크크 무료 폰트 **북크명조 / 북크고딕** 기준입니다.
> 없으면 기본 글꼴로 표시되고, 설치하면 영상과 똑같아집니다.
> (부크크 사이트에서 "북크명조", "북크고딕" 검색 → 무료 다운로드)

## 🛠️ 내 맘대로 고치기 (코드 몰라도 OK)

마음에 안 드는 부분은 **AI한테 말로** 고치세요.

- "자간 좀 더 넓혀줘"
- "폰트 고딕으로 바꿔줘"
- "강조색 파랑으로 바꿔줘"

캡처 한 장 + 이 `claude-style.css` 파일을 던지면, 클로드가 CSS를 바로 고쳐줍니다.
**VS 코드 + 클로드 코드**로 세팅해두면 실시간으로 수정·보완할 수 있습니다.
